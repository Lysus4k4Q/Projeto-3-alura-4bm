Programação

O que é uma variável em JavaScript?

Uma variável é um espaço na memória utilizado para armazenar dados ou informações temporárias.

HTML

Qual a diferença entre as tags <div> e <span> no HTML?

A tag <div> é usada para estruturar o conteúdo em blocos, enquanto a tag <span> é usada para agrupar conteúdo inline.

CSS

O que é o seletor div > p no CSS?

O seletor div > p seleciona todos os elementos <p> que são filhos diretos de um elemento <div>.
